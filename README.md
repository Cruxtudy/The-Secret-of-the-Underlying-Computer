## **💡 어떤 스터디인가요?**

> 우리는 매일 코드를 작성하지만, 정작 그 코드가 컴퓨터 내부에서 어떻게 실행되는지는 잘 모릅니다. React의 Virtual DOM이 실제 DOM보다 빠른 이유, async/await이 어떻게 동작하는지, 왜 특정 코드 패턴이 성능 문제를 일으키는지... 이 모든 질문의 답은 '밑바닥'에 있습니다.
>
> 이 스터디는 단순히 "코드가 동작하면 됐지"라는 단계를 넘어, **코드 이면에서 일어나는 일을 이해하고 더 나은 코드를 작성하고 싶은 개발자**들을 위한 여정입니다.
>
> 원하는 목표를 향해 가는 길은 언제나 험난합니다. 하지만 컴퓨터 시스템의 본질을 이해하면, 디버깅 시간은 줄고 코드 품질은 올라가며, 기술 의사결정의 근거가 명확해집니다. **총 480페이지**, 300개 이상의 그림과 일상 속 비유로 쉽게 설명된 이 책과 함께, 추상화 계층을 하나씩 벗겨내며 진짜 컴퓨터를 만나보세요.
